# DAO DAO Proposals Index

Indexes DAO DAO proposal statuses and votes. Used to get timestamps for proposal
states as well as provide data for the inbox.

# Getting Started

### 1. Install dependencies

```shell
yarn
```

### 2. Generate types

```shell
yarn codegen
```

### 3. Build

```shell
yarn build
```

### 4. Run locally

```shell
yarn start:docker
```
