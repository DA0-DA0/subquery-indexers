# DAO DAO DAOs Index

Indexes DAOs. Used to support the DAO DAO search and provide a public API for
listing DAOs.

# Getting Started

### 1. Install dependencies

```shell
yarn
```

### 2. Generate types

```shell
yarn codegen
```

### 3. Build

```shell
yarn build
```

### 4. Run locally

```shell
yarn start:docker
```
