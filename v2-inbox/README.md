# DAO DAO V2 Inbox Index

Indexes DAO DAO proposal open statuses and the wallets that vote on them. Used
to support the DAO DAO V2 Inbox page.

# Getting Started

### 1. Install dependencies

```shell
yarn
```

### 2. Generate types

```shell
yarn codegen
```

### 3. Build

```shell
yarn build
```

### 4. Run locally

```shell
yarn start:docker
```
